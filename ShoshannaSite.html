BreakBot

A Microsoft Teams chatbot to remind users to take breaks.
This bot has been created using the Bot Framework, it shows how to create a simple bot that accepts input from the user and echoes it back.

Current Features

  • Sends the user a reminder to take a break every hour (default)
  • Allow the user to customize how often they want to be reminded
  • Remind the user about the importance of taking breaks for physical and mental wellbeing
